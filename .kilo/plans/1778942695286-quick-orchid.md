# Plan: Fix GitHub OAuth Callback Route

## Root Cause Analysis
Today a hyphen/underscore mismatch in `.env` caused `redirect_uri=undefined` to be sent to GitHub. GitHub then redirected back with `?code=<something>`, and the callback route crashed with:
> "No response is returned from route handler" — receiving `undefined`

The callback route has **two logic bugs** causing that `undefined` return:

1. **Line 10 — Wrong URL protocol**: `https:/github.com` (one slash) instead of `https://github.com` (two slashes). GitHub actually redirects the malformed URL to the correct one, so here the token exchange "works" — but the fix is still needed.

2. **Lines 22–37 — Inverted early-return logic**:
   - The guard `if (!token) { return` (bare return → returns `undefined` to Next.js = **500 error**) is the *first* branch taken when there is **no** token. The success block (cookie-set + redirect) sits inside this unreachable branch.
   - When `token` **does** exist (normal success), the `if (!token)` block is skipped, execution reaches line 38, the function ends, and **nothing is returned** = another path to `undefined`/500.
   - In short: the `if` condition and the return are in the wrong branches.

3. **Response type mismatch (secondary)**: `res.json()` on GitHub's access-token response — GitHub sends `application/x-www-form-urlencoded`, not JSON. Calling `.json()` will throw in some runtimes, which also causes a 500.

## Files to Change

### `app/api/github/callback/route.ts`
Re-write the handler to fix all three issues:

```typescript
import { NextResponse } from "next/server";

export async function GET(req: NextRequest) {
    const code = req.nextUrl.searchParams.get('code');

    if (!code) {
        return NextResponse.redirect(new URL('/workspace?error=missing_code', req.url));
    }

    // Exchange code for access token
    const res = await fetch('https://github.com/login/oauth/access_token', {
        method: 'POST',
        headers: { 'content-type': 'application/json', Accept: 'application/json' },
        body: JSON.stringify({
            client_id: process.env.GITHUB_CLIENT_ID,
            client_secret: process.env.GITHUB_CLIENT_SECRET,
            code
        })
    });

    // GitHub returns form-encoded; parse it
    const text = await res.text();
    const params = new URLSearchParams(text);
    const token = params.get('access_token');

    if (!token) {
        return NextResponse.redirect(new URL('/workspace?error=no_token', req.url));
    }

    // Set token in http-only cookie and redirect to workspace
    const response = NextResponse.redirect(new URL('/workspace', req.url));
    response.cookies.set('gh_token', token, {
        httpOnly: true,
        secure: process.env.NODE_ENV === 'production',
        maxAge: 60 * 60 * 24 * 30,  // 30 days
        path: '/',
        sameSite: 'lax'
    });

    return response;
}
```
